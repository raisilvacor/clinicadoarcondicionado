# Imagens

Coloque suas imagens nesta pasta para que sejam exibidas no site.

## Imagens dos Tipos de Ar Condicionado

- `split-tradicional.jpg` - Imagem para o tipo Split Tradicional
- `split-inverter.jpg` - Imagem para o tipo Split Inverter
- `multi-split.jpg` - Imagem para o tipo Multi Split

## Imagens da Oficina

Você pode adicionar quantas imagens quiser da sua oficina:
- `oficina1.jpg` - Primeira imagem da oficina
- `oficina2.jpg` - Segunda imagem da oficina
- `oficina3.jpg` - Terceira imagem da oficina
- `oficina4.jpg` - Quarta imagem da oficina
- `oficina5.jpg` - Quinta imagem da oficina
- `oficina6.jpg` - Sexta imagem da oficina

**Nota:** O site está configurado para exibir até 6 imagens da oficina. Se você quiser adicionar mais, edite o arquivo `templates/index.html` e adicione mais elementos `.lab-image-card` na seção "Nossa Oficina".

## Formato Recomendado

- **Tipos de Ar Condicionado:** Imagens retangulares (proporção 16:9 ou 4:3), resolução mínima 800x600px
- **Oficina:** Imagens retangulares ou quadradas, resolução mínima 600x600px
- **Formato:** JPG ou PNG
